# 坦克大战 (Tank Battle)

A classic tank battle game built with HTML5 Canvas, inspired by the NES game Battle City.

## How to Play

Open `index.html` in any modern browser.

### Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move tank |
| Space | Shoot |
| P | Pause |

### Game Rules

- Destroy all enemy tanks to clear the wave
- Protect your base (★) at the bottom of the map
- Brick walls can be destroyed by bullets
- Steel walls are indestructible
- You have 3 lives — lose them all and it's game over
- Each enemy destroyed = 100 points

## Features

- Multiple map layouts
- Increasing difficulty per wave
- Enemy AI that pursues your base
- Destructible terrain
- Bullet-vs-bullet collision
